## Prerequisites

- [lerd](https://lerd.sh/)
- [pnpm](https://pnpm.io/)

## Initial setup

1. Clone the Hydrodactyl panel repository  
   `git clone https://github.com/BlueprintFramework/hydrodactyl.git`

1. Cd into the Hydrodactyl directory  
   `cd Hydrodactyl`

1. Run the dev env  
   `pnpm dev:setup`

1. Access Hydrodactyl at your new url  
   [`http://hydrodactyl.test`](http://hydrodactyl.test)
